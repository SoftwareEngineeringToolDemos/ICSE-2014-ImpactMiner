Reorganizing the concern model

It is possible to reorganize the concern model by moving and copying elements from one concern to another.

Actions:

  1. Select one or more elements in the ConcernMapper view and drag them to a different concern.

Notes:

  • By holding down the Ctrl key while dropping the selection, the elements are copied instead of moved. Because of a bug (#2), to copy an element you must first drag the element over a concern name and then press down the Ctrl key (while holding the mouse button).
  • If the selection includes elements that are neither methods or fields, the operation cannot complete.
