Modify and element's degree

Membership Degrees:

ConcernMapper provides functionnality to set different membership degrees for concern elements. This allows you to filter some of the elements from the ConcernMapper view if they are not currently of interest. As well, this feature allows plugin developpers who wish to automatically generate concern models to provide you with a degree to which the technique used to generate the concern element is certain of its membership to that the concern.

Actions:

  1. Select one or several concern elements from the ConcernMapper view.
  2. Drag the slider at the bottom of the ConcernMapper view.

Notes:

  • All the elements in the selection will be set to the selected membership degree.
  • If one or more of the elements in the selection are not methods of fields, the degree slider will be disabled.
  • When the selection contains more than one element, the slider shows a degree value of 100 for all the elements but the value will only be actually changed once you release the slider.
