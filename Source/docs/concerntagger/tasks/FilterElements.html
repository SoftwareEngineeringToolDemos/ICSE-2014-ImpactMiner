Filter Elements below a certain threshold

Actions:

  1. Select the filter button on the ConcernMapper view local toolbar to open the filter preferences dialog.
  2. The filter can be enabled or disabled with the checkbox.
  3. The threshold value can be modified by moving the slider.

All concern elements whose membership degree is set lower than the threshold will be filtered from the ConcernMapper view.

Notes:

  • The filter can also bet set through the ConcernMapper preference page.
  • Changing the filter's threshold degree will automatically set the filter to the enabled state.
