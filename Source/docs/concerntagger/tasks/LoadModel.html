Load a concern model

Load a concern model through the file's context menu

Actions:

  1. Right-click on a file with a ".cm" file extension and select Load Concerns. A progress bar indicates the state of the operation.

Notes:

  • If the model does not contain the right extension the menu option is not displayed.
  • If unsaved information exists, you will be asked to confirm the overrride.

Load a concern model by dropping the file in the ConcernMapper view

Actions:

  1. Drag a file with a ".cm" file extension over an empty area of the ConcernMapper view. A progress bar indicates the state of the operation.

Notes:

  • If the model does not contain the right extension the drop is not allowed.
  • If unsaved information exists, the user is asked to confirm the overrride.
  • The white area to the right of an element is not "empty"
