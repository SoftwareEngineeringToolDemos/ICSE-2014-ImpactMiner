Clear the ConcernMapper View

Actions:

  1. Select the Clear View button.

Notes:

  • Clears the ConcernMapper view.
  • The previously loaded concern model remains unchanged from its last saved version and is disconnected from the view.
  • Any new concerns added thereafter are part of a new concern model.
  • This action differs from manually deleting each of the concerns by the fact that a subsequent save of the model will not overwrite the previous concern model with the new changes.
